# Water Consumption Forecasting for Smart Tanker Demand Planning

## Overview

This project predicts future water consumption trends using Machine Learning techniques.

The goal of this project is to support efficient water tanker demand planning in drought-prone regions by forecasting future water usage patterns.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Jupyter Notebook

---

## Dataset

Real-world water consumption forecasting dataset from Kaggle.

---

## Features

* Data Cleaning
* Date Preprocessing
* Feature Engineering
* Linear Regression Model
* Prediction Visualization

---

## Machine Learning Workflow

1. Dataset Loading
2. Data Cleaning
3. Date Conversion
4. Feature Extraction
5. Train-Test Split
6. Linear Regression Training
7. Prediction
8. Model Evaluation
9. Visualization

---

## Evaluation Metrics

* MAE (Mean Absolute Error)
* R² Score

---

## Output

The model predicts future water consumption and visualizes actual vs predicted values using graphs.
