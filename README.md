# Water Consumption Forecasting for Smart Tanker Demand Planning

## Overview
This project predicts future water consumption using Machine Learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

## Dataset
Real-world water consumption forecasting dataset from Kaggle.

## Features
- Data Cleaning
- Date Preprocessing
- Feature Engineering
- Linear Regression Model
- Prediction Visualization

## Goal
To forecast water consumption trends that can help optimize water tanker demand planning.

## Evaluation Metrics
- MAE (Mean Absolute Error)
- R² Score
